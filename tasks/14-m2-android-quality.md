# M2 tasks — Android quality

## M2-17 — Accessibility pass

Verify labels, focus order, touch targets, contrast, dynamic text scaling and screen-reader descriptions for all primary flows.

## M2-18 — Offline/reconnect test matrix

Test airplane mode, tunnel loss, token expiry, process completion while offline, replay gap and expired approval. Each case must specify visible UI and command behavior.

## M2-19 — Notification privacy review

Verify push payloads contain identifiers only, deep links reauthorize, resolved approvals cannot be acted on, and lock-screen previews follow preference.
