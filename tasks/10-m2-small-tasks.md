# M2 decomposed tasks

## M2-08 — API error catalog

Define stable codes for unauthorized, forbidden, not found, conflict, expired approval, idempotency conflict, cursor expired and adapter unavailable. Map each to HTTP behavior.

## M2-09 — WebSocket connection manager

Implement hello/welcome negotiation, subscription authorization, heartbeat, bounded send queue and clean close reasons. Test slow and reconnecting clients.

## M2-10 — Android server list

Specify and implement the local server record, pairing entry point, connectivity status and removal flow without storing access tokens in general preferences.

## M2-11 — Android session timeline

Render durable lifecycle, approval and completion events separately from high-volume output. Support pagination and a visible resync state.

## M2-12 — Android approval detail

Show redacted action context, risk category, expiry and final state. Disable action controls when offline, expired or already resolved.
