# M4 tasks — hardening

## M4-10 — Host and proxy hardening

Implement the service account, bind/firewall, TLS, proxy limits and WebSocket timeout controls from `docs/57-deployment-hardening.md`.

## M4-11 — Secret handling audit

Search source, fixtures, logs, notifications and support bundles for credential leakage. Add regression tests for redaction patterns and notification minimization.

## M4-12 — Plugin review gate

Require manifest, capability, permission, lifecycle and security evidence before enabling any plugin outside the built-in adapter.
