# User guide outline

## First use

Install CAR server in the homelab, register a workspace, pair the Android device locally, and verify a fake-agent session before enabling remote access.

## Daily workflow

Open a workspace, resume or start a session, send a prompt, monitor output, inspect changes, and resolve approvals from the detail screen. Use timeline and event replay after reconnect rather than assuming a timeout stopped the agent.

## Safe operation

Review command category and workspace before approval. Keep direct pushes and destructive operations behind approval. Revoke a lost phone immediately from the local administration interface.
