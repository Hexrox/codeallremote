# Roadmap

## Milestone 0 — Foundation

Establish product language, architecture, ADRs, domain model, protocol requirements and implementation work packages.

## Milestone 1 — Single-user local core

Register one workspace; launch and observe a Claude Code process; persist sessions and events; provide a local authenticated API.

## Milestone 2 — Android remote control

Build Android pairing, session list, session detail, terminal stream, reconnect handling and approval push/actions.

## Milestone 3 — Secure remote access

Deploy the homelab server behind WireGuard and a VPS reverse proxy; add device revocation, audit retention, backup and monitoring.

## Milestone 4 — Operational maturity

Add diff/file views, session replay, workspace policies, Web dashboard and automated integration tests.

## Milestone 5 — Extensibility

Stabilize adapter SDK and add at least one non-Claude agent adapter.
